## 🔄 ポストフロップの違い

### プリフロップ
```
✅ 全てのアクティブポジションが選択可能
✅ 選択したポジションより前は自動Fold
✅ スキップ機能で爆速入力
```

### ポストフロップ
```
⚠️ 現在の手番のプレイヤーのみ選択可能
⚠️ 他のプレイヤーは無効化
⚠️ 順番通りにアクションを入力する必要がある
```

│  └──────┘  └──────┘  └──────┘              │

## 🎓 Tips & Tricks

### 💡 Tip 1: 連続フォールドを一瞬で入力
```
UTG～HJまで全員フォールド、COレイズの場合:
→ COを直接選択 → Raise
   （UTG, HJは自動Fold）
```

### 💡 Tip 2: プリセットボタンを活用
```
プリフロップ: 2x, 3x ボタンで即座にレイズ
ポストフロップ: 33%, 50%, 75%, 100% で素早くベット
```

### 💡 Tip 3: ヒーロー位置を意識
```
黄色で表示されているのが自分のポジション
手番の時は特に強調されるので見逃さない
```

### 💡 Tip 4: ログで確認
```
画面上部のアクションログで、
今まで何が起きたか常に確認できる
```
